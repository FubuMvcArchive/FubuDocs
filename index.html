FubuDocs

Frictionless documentation tooling

FubuDocs is a tool, heavily influenced by readthedocs.org for publishing and maintaining living technical documentation for code-centric projects. FubuDocs is built on top of FubuMVC and the Spark view engine, but it's perfectly usable in other contexts and doesn't make (m)any assumptions about your codebase. It is designed to help you keep your documentation right alongside your actual code so that nothing gets out of sync.

To make hosting and deployment simple, it exports all of the rendered documentation into plain ol' static html. Host it on GitHub pages, dropbox, wherever you'd like.

Feature List

Some highlights


  • Simple Hosting

    The end result is just simple, static html. You can host it anywhere you like.


  • Powerful tooling

    Your documentation is written with all of the power of FubuMVC behind it. Live reloading, live code snippets from your sample projects/unit tests, and many other tools come for free.


  • Clean markup

    Using the Spark view engine and its built-in markdown capabilities, your markup is clean and easy to maintain. Combine that with the power of Spark's bindings and all of the defaults from FubuDocs and you can focus on your content rather than the lack of tooling.


  • Responsive Design

    FubuDocs sports a clean and responsive layout that works great with most devices.
