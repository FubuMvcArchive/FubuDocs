Gathering Code Snippets

fubudocs snippetsScrapes the entire solution for files with code snippets and places those files under the '/snippets' directory of the documentation project

fubudocs snippets[-s, --scan] [-q, --quiet] [-l, --list] [-d, --directory ]

Flags
[-s, --scan]Scans and lists codesnippets, but does not perform the import
[-q, --quiet]Turns off most of the tracing messages
[-l, --list]Finds and lists all the existing code snippets, but does not execute anything
[-d, --directory <directory>]The directory holding the docs. Will try to find a single directory containing the name 'Docs' under an 'src' folder if this flag is not specified
